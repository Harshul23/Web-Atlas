**Your Complete Journey of Web Development in One Place**  

From **class notes** → **hands-on exercises** → **real projects**, WebAtlas is an open-source hub that documents everything I learn in web development at Vedam School of Technology.  

---

## ✨ Features  
- 📖 **Notes** — Simplified explanations of web dev topics  
- 📝 **Exercises** — Practice problems with solutions  
- 💻 **Projects** — Real HTML/CSS/JS projects you can run & learn from  
- ⚡ **Dynamic Structure** — Notes and projects load from JSON (no endless `.html` files)  

---

## 📂 Project Structure  
```
WebAtlas/
 ├── index.html        # Homepage  
 ├── notes.html        # Dynamic notes page  
 ├── projects.html     # Interactive project gallery  
 ├── data/             # JSON files for notes & projects  
 ├── js/               # Scripts to load content dynamically  
 ├── css/              # Stylesheets  
 └── projects/         # Folder containing project files  
```

---

## 🚀 Getting Started  

### 1. Clone the repo  
```bash
git clone https://github.com/your-username/WebAtlas.git
cd WebAtlas
```

### 2. Open in browser  
Since it’s a **static site**, just open `index.html` in your browser.  

### 3. Explore  
- Go to **Notes** for structured lecture notes  
- Go to **Projects** to see live demos of web dev projects
