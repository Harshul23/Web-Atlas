UNIT – 2: CSS BASICS & SYNTAX

1. CSS Syntax

−

CSS syntax controls how styles are written and applied. It consists of a selector (which element to style) and a declaration block (style rules written inside curly braces { } ). Each declaration contains a property and a value . Correct syntax ensures browsers interpret styles properly.

CSS syntax is simple, readable, and works consistently across HTML documents.

Syntax Diagram :

selector {
  property: value;
  property: value;
}

Example :

p {
  color: blue;
  font-size: 18px;
}

2. Application Methods (Inline, Internal, External)

+

CSS can be applied in three main ways. Inline CSS is written directly in the HTML element and is useful for very small changes but not recommended for large projects. Internal CSS is placed inside a <style> tag within the <head> section—good for single‑page styling. External CSS is stored in a separate .css file linked using the <link> tag, which keeps code cleaner and more reusable.

Inline CSS :

<p style="color: red;">Hello</p>

Internal CSS :

<style>
  h1 { color: green; }
</style>

External CSS :

<link rel="stylesheet" href="style.css">

3. CSS Selectors (Element, Class, ID, Pseudo)

+

Selectors determine which HTML elements will receive CSS. They help target specific parts of the page. Element selectors style all tags of that type. Class selectors style multiple elements by assigning a reusable name. ID selectors target one unique element. Pseudo‑classes target states such as hover, focus, active, first‑child, etc. Proper selectors make CSS organized and efficient.

Examples :

/* Element */
h1 { color: blue; }

/* Class */
.box { padding: 20px; }

/* ID */
#header { background: black; }

/* Pseudo-class */
button:hover { background: orange; }

4. CSS Box Model

+

Every HTML element behaves like a box. The box model defines the structure: content , padding , border , and margin . Understanding this model is crucial for layout design, spacing, and alignment. It helps prevent overlapping and allows precise control of element sizes.

Example :

.box {
  margin: 20px;
  padding: 15px;
  border: 2px solid black;
}

5. Margin, Padding & Border

+

Margin provides outside spacing, padding gives inside spacing, and border surrounds an element. These three properties shape the layout and ensure readability. They help organize UI components, separate sections, and maintain a proper visual structure on screens.

Code :

.card {
  margin: 20px;
  padding: 15px;
  border: 1px solid gray;
}

6. Text & Font Styling

+

Text and font properties enhance readability and design. CSS allows setting font-size, font-family, font-weight, line-height, text-align, letter-spacing, and decoration. These styles help create visually appealing headings, paragraphs, and user interfaces.

Code :

p {
  font-size: 18px;
  font-family: Arial;
  text-align: justify;
}

7. CSS Backgrounds

+

Background properties help customize the visual appearance of elements using colors, gradients, or images. They enhance design quality and are commonly used in banners, sections, hero pages, and cards. With background-size and repeat options, designers can create professional and attractive UI sections easily.

Code :

.box {
  background-image: url("pic.jpg");
  background-size: cover;
}

8. Borders & Border Radius

+

Borders help visually define boundaries of boxes, while border-radius creates rounded corners for a smoother design. These properties enhance the aesthetics of buttons, cards, images, and containers. They play an important role in modern minimalistic design styles.

Code :

.box {
  border: 2px solid green;
  border-radius: 10px;
}

9. Layout with Flexbox

+

Flexbox is a one-dimensional layout system used to align items horizontally or vertically. It adapts to space automatically, making responsive design easy. Flexbox removes difficulties of old layout methods and provides powerful alignment control.

Four Important Flexbox Properties :

  1. display: flex – activates flexbox on the container. 
    .container { display: flex; }
  2. justify-content – aligns items horizontally (main axis). 
    .container { justify-content: center; }
  3. align-items – aligns items vertically (cross axis). 
    .container { align-items: center; }
  4. flex-direction – sets the direction of items. 
    .container { flex-direction: column; }

10. Layout with Grid

+

CSS Grid is a two-dimensional layout system used to design full webpage structures with rows and columns. It allows precise control over areas and is used for dashboards, galleries, and full layouts.

Example :

.grid {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
  gap: 20px;
}

11. Hover and Focus Styles

+

Pseudo-classes like :hover and :focus create interactivity. :hover triggers when the mouse is over an element, while :focus triggers when input fields or interactive elements are active. These are widely used for buttons, menus, and forms.

Example :

button:hover {
  background: red;
}

input:focus {
  border-color: blue;
}

12. Responsiveness in CSS

+

Responsiveness ensures a webpage looks good on all screen sizes—desktop, tablet, and mobile. CSS automatically adjusts layout, font size, images, and spacing based on device width. Techniques such as media queries, percentage widths, flexible units, flexbox, and grid help achieve responsiveness. A responsive site improves user experience and accessibility.

13. Media Queries

+

Media queries allow CSS to apply different styles depending on screen size. They are essential for responsive design because they let layouts adjust for mobile, tablet, and desktop screens. Media queries can detect device width, orientation, resolution, and more.

Example :

@media (max-width: 600px) {
  .box {
    width: 100%;
    background: lightblue;
  }
}
